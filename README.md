# BirdBomber

A 2D game prototype.

## Requirements

* Unity 5.x.x

## Clone and Run

Clone into BirdBomber directory.

```
$ git clone --branch develop <url/to/birdbomber.git> BirdBomber
```

After cloning, open and run with Unity.

## How to Play

Use arrow keys to move. Use space key to drop bomb.
